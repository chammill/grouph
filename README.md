# S.E.E.D.
To play S.E.E.D. (Societal Evaluation Energy Drive), run test.py

# Controls
- WASD - Move 
- SPACE - Interact with objects and go through Dialogue
- LEFT MOUSE-CLICK - Switch Levers on and off
